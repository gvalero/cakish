# Changelog

Changes for range HEAD^

## ✨ Features

- Extended `metadata` export with Open Graph, Twitter Card, canonical URL, extensive keywords, robots, and geo tags in `app/layout.tsx` for enhanced SEO and social sharing.
- Added JSON-LD structured data for the `Bakery` business using a `<script type="application/ld+json">` tag in the `<head>` of `app/layout.tsx`, improving SEO with schema.org rich results.
- Updated `metadata` export in `app/order/page.tsx` to include a more descriptive `title`, expanded `description`, and added `alternates.canonical` for improved SEO and canonical URL handling.
- Updated `metadata` export in `app/story/page.tsx` with a more descriptive `title`, expanded `description`, and added `alternates.canonical` for better SEO and canonical URL configuration.
- Added an "Under Construction" banner at the top of the site header in `components/site-header.tsx` to notify users that the site is coming soon.
- Added `public/robots.txt` to allow all user agents, disallow `/v1/` and `/test-payment/`, and reference `https://cakish.ie/sitemap.xml` for improved search engine crawler management.
- Added `public/sitemap.xml` for basic site indexing, listing `/`, `/order/`, and `/story/` with change frequency and priority for SEO.

## 🐛 Bug Fixes

- Changed hero tagline in `app/page.tsx` from "Handcrafted in Dublin" to "Handcrafted in Wicklow" to accurately reflect product origin.
